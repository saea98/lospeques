# San Valentín - Mensajes Personalizados

Web para compartir mensajes de San Valentín personalizados.

## Con Docker

**Construir la imagen:**
```bash
docker build -t san-valentin .
```

**Ejecutar el contenedor:**
```bash
docker run -p 8000:8000 san-valentin
```

O con docker-compose:
```bash
docker-compose up --build
```

Luego abre http://localhost:8000
